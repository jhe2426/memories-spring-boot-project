## 프로젝트 개요
### Memories Back

## REST API 명세서
### 링크
배포 링크 : https://github.com/jhe2426/cognitive-document

## 주요 기능
- 회원가입, 로그인
- 카카오, 네이버 로그인
- 기억력 검사
- 집중력 검사
- 일기 작성, 수정, 삭제, 보기
- Chat GPT API 활용 기억력 및 집중력 향상 방법 제공

## 기슬 스택
- Spring boot 3.5.5
- Java 23
- Lombok
- Spring web
- Spring validation
- Spring data JPA
- Spring security
- Spring oauth2 client
- JJWT 0.11.2
- MySQL Connector
